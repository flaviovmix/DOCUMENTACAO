Array (Vetor)

0:00 / 0:00

Um array (ou vetor) é uma coleção de valores do mesmo tipo, guardados em sequência e acessíveis por um número de posição chamado índice. Pense numa estante com 10 prateleiras numeradas de 0 a 9: cada prateleira guarda um valor, e você acessa qualquer uma pelo número dela.

public class Main {
public static void main(String[] args) {
int[] vetorzao = new int[10];
vetorzao[0] = 3;
vetorzao[1] = 4;
vetorzao[3] = 13;
vetorzao[2] = 8;
vetorzao[5] = 51;
vetorzao[4] = 22;
vetorzao[6] = 33;
System.out.println("Primindo...");
for (int i = 0; i < vetorzao.length; i++) {
System.out.println(vetorzao[i]);
}
System.out.println("Até a próxima aula");
}
}

Tamanho fixo — declarado na criação +

Quando você escreve new int[10], o Java reserva exatamente 10 posições contíguas na memória. Esse tamanho não muda durante a vida do array — não dá pra "acrescentar uma prateleira" depois.

int[] v = new int[10];
v[10] = 99;  // ArrayIndexOutOfBoundsException — só existe [0]..[9]

Se você precisa de uma coleção que cresce conforme adiciona itens, use ArrayList (próxima trilha). Array nu é o tijolo mais básico — rápido e simples, mas inflexível.

Índice começa em 0 — a pegadinha clássica +

A primeira posição é [0], não [1]. Num array de tamanho 10, os índices válidos vão de 0 até 9.

ÍndiceSignificado
v[0]Primeira posição
v[1]Segunda posição
v[3]Quarta posição (não a terceira!)
v[9]Última posição (em um array de tamanho 10)
v[10]❌ Erro — fora dos limites

E a ordem das atribuições não importa: o código preenche [3] antes de [2], e o array fica certinho do mesmo jeito. Cada posição é endereçável de forma independente.

Valor default — posição vazia não é nula, é zero +

Quando o Java cria um array de int, todas as posições já nascem com 0. Você não precisa preencher nada antes de usar.

É por isso que a animação acima imprime 0 0 0 nas três últimas posições: a gente atribuiu valor só nos índices 0 a 6, então 7, 8 e 9 continuam com o default.

Iterando com for — o padrão clássico +

Pra percorrer todas as posições, o padrão é o for com contador:

for (int i = 0; i < vetorzao.length; i++) {
    System.out.println(vetorzao[i]);
}

3 partes:

  • int i = 0 — começa do índice 0 (primeira posição)
  • i < vetorzao.length — vai até o último índice válido. .length é uma propriedade do array (sem parênteses, não é método)
  • i++ — avança 1 posição por iteração

⚠️ Atenção: a condição é i < length, não i <= length. Se o tamanho é 10, o último índice válido é 9 — usar <= faria estourar pra v[10].

Versão moderna: for-each

Quando você só precisa do valor (e não do índice), dá pra escrever mais limpo:

for (int v : vetorzao) {
    System.out.println(v);
}

Lê assim: "pra cada v dentro de vetorzao...". Atalho introduzido no Java 5.
