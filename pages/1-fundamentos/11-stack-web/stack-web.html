STACK WEB

As peças que montam uma aplicação web moderna — Svelte e React no frontend, Spring no backend

Framework de front reativo — componentes `.svelte`, runes ($state, $derived, $effect) e o compilador que vira JS puro sem virtual DOM.

SVELTE

Biblioteca de UI baseada em componentes — JSX, hooks (useState, useEffect) e virtual DOM; a base do Next.js e do ecossistema front mais popular.

REACT

Framework Java pra back-end — injeção de dependência, controllers REST, JPA e o ecossistema todo pré-configurado pelo Spring Boot.

SPRING

Onde a aplicação roda — Tomcat, PostgreSQL, Tailscale, Cloudflare, Docker; o que precisa ficar de pé pra Svelte/React/Spring funcionarem.
