Tela de listagem de Produto. Exibe todos os registros em uma ListView simples — sem paginação, sem ordenação por coluna, sem exclusão inline. É a versão mínima de uma listagem funcional no framework Jasap.

CÓDIGO COMPLETO 
package br.xt.app.departamento.produto;

import br.jasap.core.Effect;
import br.jasap.effect.Response;
import br.jasap.gui.JasapPage;
import br.jasap.gui.ListColumn;
import br.jasap.gui.ListLine;
import br.jasap.gui.ListView;
import br.jasap.gui.Table;
import br.xt.acore.view.XtPage;

public class DepartamentoProdutoList extends DepartamentoProdutoAction {

    @Override
    public Effect execute() throws Exception {
        render();
        return new Response();
    }

    private void render() throws Exception {
        XtPage page = new XtPage(getManager());
        if (isAjaxCall()) {
            update(JasapPage.DIV_WINDOW, page.content(window().toHtml()));
        } else {
            page.getTable()
                .setBorder(4)
                .rowC("100%")
                .setContent(page.content(window().toHtml()))
                .setStyle(ui().stretchBorder());
            page.setWinTitle("Produtos");
            getOutput().write(this, page);
        }
    }

    public Table window() throws Exception {
        Table w = new Table(getManager()).setSize("100%", "100%");
        w.rowC("100%", JasapPage.DIV_WSPACE, lView());
        return w;
    }

    private ListView lv = null;
    public ListView lView() throws Exception {
        if (lv == null) {
            lv = ui().lView();
    
            getFactory().departamento().proModel().daoList(lv.getData());
    
            ListColumn col_nome = lv.newColumn("Nome").setWidth(220).setPadding(";padding:10 8 10 8;");
            ListColumn col_vl   = lv.newColumn("Valor").setWidth(100).setPadding(";padding:10 8 10 8;").alignCenter();
            ListColumn col_qtd  = lv.newColumn("Qtd").setWidth(80).setPadding(";padding:10 8 10 8;").alignCenter();
            ListColumn col_obs  = lv.newColumn("Observação").setPadding(";padding:10 8 10 8;");
    
            while (lv.hasNext()) {
                DepartamentoProdutoBean bean = (DepartamentoProdutoBean) lv.next();
                ListLine line = lv.createLine();
    
                col_nome.setContent(bean.getNome_pro());
                col_vl.setContent(bean.getVl_pro());
                col_qtd.setContent(bean.getQtd_pro());
                col_obs.setContent(bean.getObs_pro());
    
                lv.addLine(line);
            }
        }
        return lv;
    }

}
execute() e render()

execute() é o ponto de entrada — apenas chama render() e retorna.

render() decide como entregar a tela ao browser:

ContextoO que acontece
Chamada Ajax (ex: abertura via modal)update(DIV_WINDOW, ...) — substitui o conteúdo do DIV_WINDOW sem recarregar a página inteira
Acesso direto pelo browser (URL)getOutput().write(this, page) — escreve o HTML completo com CSS, JS e título da janela

Na prática, 99% das vezes a lista abre via Ajax (pelo MenuItem do Home). O else é um fallback para o caso raro de acesso direto pela URL.

Table aqui é um componente de layout do framework — não é uma tabela HTML de dados. Nesta versão mínima, a tela tem uma única linha:

ParâmetroO que faz
"100%"A ListView ocupa toda a altura disponível
JasapPage.DIV_WSPACEID do div — permite scroll interno quando há muitos registros
lView()O conteúdo — a ListView com os dados
lView() — a ListView

Inicialização lazy

Mesmo padrão do proBean() na Action: o componente só é criado na primeira chamada e reutilizado nas seguintes dentro da mesma requisição.

daoList

getFactory().departamento().proModel().daoList(lv.getData()) executa a consulta no banco e preenche a lista interna da ListView. Nesta versão mínima não há filtros — lista todos os registros.

Colunas

Observaçãolivre (ocupa o restante)esquerda (padrão)

Nesta versão mínima, as colunas não têm setOrderBy() — não são clicáveis para ordenar.

Loop de registros

while (lv.hasNext()) itera sobre todos os registros retornados. Para cada um:

  • lv.next() avança e retorna como Object — cast para DepartamentoProdutoBean
  • lv.createLine() cria uma linha (sem setOnclick — não é clicável nesta versão)
  • setContent(valor) preenche cada célula — escapa HTML automaticamente
  • lv.addLine(line) adiciona a linha à ListView
