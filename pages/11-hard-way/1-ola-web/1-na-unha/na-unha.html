A página na unha

0:00 / 0:00

O primeiro passo da Fase 1, antes de qualquer framework: montar uma página que reage ao clique usando só HTML e JavaScript puro, mexendo no DOM com as próprias mãos. O objetivo não é o resultado (um contador e uma lista) — é sentir na pele o trabalho de manter a tela e os dados em sincronia. Essa dor é o que vai dar sentido ao Svelte lá na frente.

O que vamos fazer — e por quê +

Uma página com duas coisas que mudam sozinhas na tela quando você interage:

  • um contador: um número e um botão "+1" que soma 1 a cada clique;
  • uma lista de itens, cada um com um botão "remover".

Sem framework, quem faz isso acontecer na tela é você, escrevendo JavaScript que mexe no DOM na mão. Guarde uma pergunta pro fim: quantas coisas eu preciso lembrar de fazer toda vez que algo muda?

O HTML mínimo — só o esqueleto +

O HTML aqui é só a estrutura: um número, um botão e uma lista vazia. Repare nas etiquetas id — elas servem pro JavaScript achar cada elemento depois.

<body>
    <h1>Olá Web na unha</h1>

    <p>Contador: <span id="numero">0</span></p>
    <button id="botao">+1</button>

    <h2>Lista</h2>
    <ul id="lista"></ul>

    <script>
        // o JavaScript entra aqui
    </script>
</body>

O <span id="numero">0</span> é onde o número aparece (começa em 0). O <ul id="lista"> nasce vazio — quem vai preencher é o JavaScript. Por enquanto o botão não faz nada: o cérebro dele é o próximo passo.

O contador em JavaScript — o DOM na mão +

Esse é o coração da fase. Vamos por partes.

let contador = 0;

const numero = document.getElementById("numero");
const botao = document.getElementById("botao");

botao.addEventListener("click", function () {
    contador = contador + 1;       // 1) muda o estado
    numero.textContent = contador; // 2) ATUALIZA a tela na mão
});

Variáveis: caixinhas com nome

Uma variável é uma caixa com nome que guarda um valor. let cria uma caixa que pode mudar (o contador vai virar 1, 2, 3…). const cria uma caixa que não troca de conteúdo (a referência ao elemento da tela é sempre a mesma). Regra prática: use const por padrão, let só quando o valor vai mudar.

O DOM e o document

Quando o navegador lê seu HTML, ele monta uma árvore de elementos na memória — isso é o DOM. O document é o galho-raiz: representa a página inteira, e por ele o JavaScript alcança qualquer pedaço da tela.

getElementById: achar um elemento

document.getElementById("numero") diz: "ó document, me acha o elemento com id="numero"". Ele devolve uma referência àquele <span>, que guardamos na caixinha numero pra usar depois.

textContent: mudar o que aparece

textContent é o texto dentro de um elemento. Escrever nele (com =) troca o que aparece na tela. numero.textContent = contador = "pegue o número atual da memória e pinte na tela".

addEventListener: reagir ao clique

addEventListener("click", ...) quer dizer: "quando este botão for clicado, rode este bloco". O function () { ... } é uma função — um pacote de instruções guardado pra rodar depois (a cada clique). Dentro dela são duas ações: soma 1 no estado, e depois pinta na tela. Esqueça a segunda linha e o número muda na memória mas a tela congela.

A lista em JavaScript — construindo e limpando +

Aqui a dor cresce: a cada mudança você apaga a lista inteira e redesenha do zero.

let itens = ["Maçã", "Banana", "Laranja"];
const lista = document.getElementById("lista");

function desenharLista() {
    lista.innerHTML = ""; // apaga a lista inteira...
    for (let i = 0; i < itens.length; i++) {
        const li = document.createElement("li");
        li.textContent = itens[i] + " ";

        const remover = document.createElement("button");
        remover.textContent = "remover";
        remover.addEventListener("click", function () {
            itens.splice(i, 1); // tira o item do array
            desenharLista();    // ...e redesenha tudo de novo, na mão
        });

        li.appendChild(remover);
        lista.appendChild(li);
    }
}

desenharLista(); // desenha a primeira vez

Array: uma lista de valores

Um array é uma lista ordenada de valores numa caixa só. Cada posição tem um índice que começa em 0: itens[0] é "Maçã". itens.length é quantos tem (3). itens.splice(i, 1) = "a partir da posição i, remova 1 item" — como tirar uma carta do meio do baralho.

for: repetir pra cada item

O for repete um bloco. Lê-se: comece com i = 0; enquanto i for menor que o tamanho da lista; depois de cada volta, i++ (soma 1). Resultado: o bloco roda uma vez pra cada item.

createElement e appendChild: montar a árvore

createElement("li") fabrica um elemento novo na memória. appendChild pendura um elemento dentro de outro (o botão dentro do <li>, o <li> dentro da <ul>). E innerHTML = "" esvazia um elemento. É por isso que a desenharLista() existe: a cada mudança ela apaga tudo e remonta <li> por <li>. Você é o pedreiro.

A dor — você é o motor da tela +

Resumindo o que doeu: a tela nunca se atualiza sozinha. Cada mudança no estado exige que você, na mão, mande pintar de novo.

  • No contador: mudar a variável e escrever no textContent — duas linhas.
  • Na lista: mexer no array e chamar desenharLista() — e a desenharLista redesenha a lista inteira toda vez.

Com só somar e remover já dá trabalho. Imagine adicionar ordenar, editar, filtrar — o "motor" cresce, e cada lugar onde você esquecer de atualizar a tela vira um bug silencioso. Multiplique por uma aplicação inteira: é a vida sem framework.

Essa é a dor que a próxima parte (o alívio: Svelte) vai resolver. Lá você vai declarar o estado e o framework cuida da tela — e aquela "segunda linha" obrigatória simplesmente some.

Código completo — index.html +

Tudo junto, pronto pra copiar e abrir no navegador:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Olá Web na unha</title>
</head>
<body>
    <h1>Olá Web na unha</h1>

    <p>Contador: <span id="numero">0</span></p>
    <button id="botao">+1</button>

    <h2>Lista</h2>
    <ul id="lista"></ul>

    <script>
        let contador = 0;

        const numero = document.getElementById("numero");
        const botao = document.getElementById("botao");

        botao.addEventListener("click", function () {
            contador = contador + 1;
            numero.textContent = contador;
        });

        let itens = ["Maçã", "Banana", "Laranja"];
        const lista = document.getElementById("lista");

        function desenharLista() {
            lista.innerHTML = "";
            for (let i = 0; i < itens.length; i++) {
                const li = document.createElement("li");
                li.textContent = itens[i] + " ";

                const remover = document.createElement("button");
                remover.textContent = "remover";
                remover.addEventListener("click", function () {
                    itens.splice(i, 1);
                    desenharLista();
                });

                li.appendChild(remover);
                lista.appendChild(li);
            }
        }

        desenharLista();
    </script>
</body>
</html>
