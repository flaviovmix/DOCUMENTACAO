VISAO GERAL

HTML e JavaScript puro: um contador e uma lista que mudam na tela mexendo no DOM na mão. O foco é sentir a dor de manter estado e tela em sincronia.

A PÁGINA NA UNHA

Versionar com git (init, commit) e publicar no GitHub. Inclui a pegadinha do README e as duas formas de conectar: HTTPS e SSH.

MANDAR PRO GITHUB

O scaffold, o npm run dev e o mesmo contador reativo. A reatividade mata a dor: você cuida do estado, o framework cuida da tela.
