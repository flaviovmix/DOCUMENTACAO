A base das tecnologias: anatomia de um projeto, framework vs biblioteca

O primeiro projeto do Stack Web: um "Olá Mundo" em SvelteKit, semente do app que vai crescer. Passeio arquivo por arquivo pelo que o gerador criou.

ANATOMIA DO PROJETO SVELTE

O mesmo "Olá Mundo" do Stack Web, antes em SvelteKit, agora em React. A tela é idêntica; o foco é a comparação: React é uma biblioteca que só desenha a tela (precisa do Vite em volta), enquanto o SvelteKit é um framework montado. JSX, componente como função, e a página ligada na mão no "root".
